# Cafe Management System

Python-based application for order handling, inventory checks, and automated billing.

## Features
- Create and manage menu items
- Track inventory levels and alerts
- Process orders with tax/discount rules
- Generate bills (console or file output)

## Tech Stack
- Python 3.x
- Optional: SQLite for data persistence
